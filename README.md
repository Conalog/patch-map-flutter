# patch_map_flutter

`patch_map_flutter` 패키지는 현재 초기 세팅 단계이며, 제공 기능은 준비 중입니다.

## Getting Started

```bash
flutter pub add patch_map_flutter
```

## Development

```bash
./tool/verify.sh
```

## Publish Checklist

1. Update `version` in `pubspec.yaml`
2. Update `CHANGELOG.md`
3. Run `./tool/verify.sh`
4. Run `flutter pub publish --dry-run`
5. Publish with `flutter pub publish`
